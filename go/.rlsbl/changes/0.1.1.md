## 0.1.1

### Breaking

- **Breaking: mutex groups always required.** Removed the `Required` field from `MutexGroup`. All mutex groups now require exactly one flag.

### Features

- **Flag dependencies.** New `CoRequired` and `Requires` types for declaring flags that must appear together.

### Fixes

- **Choices on global flags.** Global flags now validate choices correctly.
- **Repeatable global flags.** Global flags with `Repeatable()` now work correctly.
- **Global flag parsing fix.** `extractGlobalFlags` now correctly stops at the first non-flag token.
- **Global flag collision check.** Group commands now detect flag name collisions with global flags.
- **Choices nil slice fix.** `Choices()` no longer panics when called with no arguments.
